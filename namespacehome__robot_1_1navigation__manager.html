Detailed Description

Navigation Manager Node.

Manages sequential or random waypoint navigation for the robot. This node loads waypoints from a YAML file and navigates through them using Nav2 BasicNavigator.

Parameters: waypoints_file (str): Path to waypoints YAML file mode (str): Navigation mode - 'sequential' or 'random'
