Detailed Description

Sequential/Random Waypoint Patrol Node.

This script navigates a robot through a list of waypoints either sequentially or in random order. It handles interruptions via SIGTERM/SIGINT signals and publishes events when waypoints are reached for voice announcements.

Usage: ros2 run home_robot execute_patrol [–random]
