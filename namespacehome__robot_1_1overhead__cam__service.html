Detailed Description

Overhead Camera Service Node - Opens rqt_image_view for ceiling camera feed.

This node provides a ROS2 service to launch an image viewer window displaying the overhead camera feed. Prevents duplicate windows.

Service: /overhead_cam (std_srvs/srv/Trigger) - Open camera viewer window

Parameters: topic (string): Image topic to subscribe (default: /overhead_cam)

Usage: ros2 run home_robot overhead_cam_service ros2 service call /overhead_cam std_srvs/srv/Trigger
