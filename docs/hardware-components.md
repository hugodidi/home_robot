    <img src="../assets/schematic.png" alt="schematic" width="700" />

The hardware is composed of:

- Arduino Uno
- 8x8 DotMatrix (With MAX7219 driver)
- Buzzer (Passive buzzer)
- B103 potentiometer (10k ohm)
